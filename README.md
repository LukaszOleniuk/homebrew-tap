# Homebrew Tap for CrabPad

[CrabPad](https://crabpad.app) is a fast, premium Markdown editor for macOS, Windows, and Linux. Local-first, keyboard-driven, distraction-free.

## Install

```bash
brew install --cask LukaszOleniuk/tap/crabpad
```

Or add the tap first:

```bash
brew tap LukaszOleniuk/tap
brew install --cask crabpad
```

## Update

```bash
brew upgrade --cask crabpad
```

## Uninstall

```bash
brew uninstall --cask crabpad
```
